Prompt: 写一个Python函数，计算斐波那契数列的第n项
Response: 斐波那契数列是一个经典的递归问题，它的定义是：F(n) = F(n-1) + F(n-2)，其中F(1) = 1，F(2) = 1。下面是一个使用递归方法计算斐波那契数列的第n项的Python函数：

```python
def fibonacci(n):
    if n == 1 or n == 2:
        return 1
    else:
        return fibonacci(n-1) + fibonacci(n-2)
```

需要注意的是，递归方法存在重复计算的问题，可以使用动态规划的方法来避免重复计算，提高效率。下面是一个使用动态规划方法计算斐波那契数列的第n项的Python函数：

```python
def fibonacci(n):
    if n == 1 or n == 2:
        return 1
    fib = [0] * (n + 1)
    fib[1] = 1
    fib[2] = 1
    for i in range(3, n + 1):
        fib[i] = fib[i-1] + fib[i-2]
    return fib[n]
```
